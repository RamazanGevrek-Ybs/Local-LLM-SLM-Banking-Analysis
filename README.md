# Local-LLM-SLM-Banking-Analysis
Bankacılık sektörü için yerel (Edge) yapay zeka modellerinin veri gizliliği ve performans analizi test kodları.
bankacılık ve finans sektöründe veri gizliliğini korumak için test edilen yerel (Local/Edge) yapay zeka modellerinin (Llama-3, Qwen, Phi-3) algoritma scriptlerini, TF-IDF puanlama algoritmalarını ve veri görselleştirme kodların vardır. Makalede yer darlığı sebebiyle eklenemeyen İngilizce test senaryoları ve ham CSV logları da bu depoda açık kaynak olarak sunulmuştur.
